# Product

## Purpose

Represents products used within the humanoid robotics ecosystem.

## Required Fields

- Product Name
- Product Type
- Manufacturer
- Description
- Website
- Related Technologies
- Related Robots

## Optional Fields

- Product Version
- Release Date
- Price
- Documentation
- Notes

## Example

Product Name: NVIDIA Jetson AGX Orin

Product Type: AI Computing Module

Manufacturer: NVIDIA

Description:
High-performance AI computing platform for robotics and edge AI.

Website:
https://www.nvidia.com/

Related Technologies:
- Edge AI
- Computer Vision

Related Robots:
- Unitree G1
- Figure 02
