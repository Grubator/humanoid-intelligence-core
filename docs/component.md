# Component Entity

## Purpose

A Component is a physical part used to build a humanoid robot.

## Examples

- Battery
- Actuator
- Electric Motor
- Harmonic Drive
- Cycloidal Gearbox
- Camera
- LiDAR
- Microphone
- Force Sensor
- IMU
- Tactile Sensor
- GPU
- CPU
- Hand
- Arm
- Leg

## Required Fields

- Name
- Category
- Manufacturer
- Country
- Specifications

## Relationships

- Robots
- Technologies
- Manufacturers
- Companies
