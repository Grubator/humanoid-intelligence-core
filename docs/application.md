# Application Entity

## Purpose

An Application describes where a humanoid robot performs useful work.

## Examples

- Manufacturing
- Logistics
- Healthcare
- Hospitality
- Agriculture
- Construction
- Defence
- Home Assistance
- Retail
- Education

## Relationships

- Robots
- Manufacturers
- Technologies
- Companies
- Countries

| Field | Type | Required |
|-------|------|----------|
| name | string | Yes |
| slug | string | Yes |
| category | taxonomy | Yes |
| industry | taxonomy | No |
| description | text | No |
