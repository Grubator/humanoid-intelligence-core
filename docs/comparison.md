# Comparison

## Purpose

Represents structured comparisons between humanoid robots, technologies, manufacturers or components.

## Required Fields

- Comparison Title
- Comparison Type
- Description
- Compared Items
- Comparison Criteria
- Winner (optional)

## Optional Fields

- Score
- Advantages
- Disadvantages
- Conclusion
- Notes

## Example

Comparison Title:
Unitree G1 vs Figure 02

Comparison Type:
Robot

Description:
Comparison of two general-purpose humanoid robots.

Compared Items:
- Unitree G1
- Figure 02

Comparison Criteria:
- Price
- Speed
- Payload
- Dexterity
- AI
- Availability

Conclusion:
Each robot targets a different market segment.
