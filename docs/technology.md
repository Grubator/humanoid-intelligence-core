# Technology Entity

## Purpose

A Technology is a reusable capability used by one or more humanoid robots.

## Examples

- Vision AI
- SLAM
- Force Feedback
- Tactile Skin
- Reinforcement Learning
- Hydraulic Actuation
- Electric Actuation
- Tendon-driven Hands
